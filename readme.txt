Instructions
============
  TXMLSerializer is tested with Delphi 6. Please report it when you have another
  version of Delphi that works with the TXMLSerializer component.

  Installation:
    - Start Delphi
    - Click the menu Component, Install Component
    - Point to the XMLSerializer.pas file
    - Click OK and recompile your user package

Use
===
  All properties are available through the Object Inspector. Saving/Loading
  objects into a file can be done through the public functions.
